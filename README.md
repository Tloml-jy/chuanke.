> 一个面向大学生的校园电商平台，连接学生用户与校内认证商户，支持商品购买与服务预约。

## ✨ 核心痛点与解决方案

*   **痛点1**：UI设计到代码还原度低（传统方式<70%）。
    *   **解决**：基于我构建的一套多Agent协同系统，设计到代码的还原度提升至**85%以上**。
*   **痛点2**：小程序页面繁多（本项目达42个），开发周期长。
    *   **解决**：利用AI Agent进行**长链推理**和自动化页面生成，开发周期从14天缩短至**8小时**。
*   **痛点3**：切换视觉风格相当于重做整个项目。
    *   **解决**：抽象出CSS变量文件，实现“风格热插拔”，风格切换时间从2周缩短至**30分钟**。

## 🛠️ 技术栈与架构亮点

本项目构建了一套**多智能体协同（Multi-Agent Collaboration）的全自动设计与工程化交付系统**。

*   **视觉逆向推理Agent**：输入参考图（如喜茶），自动提取12维设计令牌（配色、圆角、阴影等）。
*   **令牌传播与冲突检测Agent**：将设计令牌注入42个页面，自动校验，确保风格一致。
*   **多模态转换Agent**：将设计稿（Figma/妙多）通过MCP协议转换为可运行的WXML/WXSS/JS代码。
*   **风格迁徙Agent**：通过“设计母版断点续传”策略，解决AI上下文窗口溢出问题，实现一键换肤。
